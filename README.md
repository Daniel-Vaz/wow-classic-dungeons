# WoW Classic — Dungeons Compendium

A single-page reference for World of Warcraft Classic dungeon quests, encounters, and loot.

## Features

- **Quest tracking** — Browse every dungeon quest chain with prerequisites, start/end NPCs, objectives, and rewards. Mark quests complete and filter by status (completed/incomplete) or by quests that award gear.
- **Encounters & loot** — View dungeon bosses and their notable item drops.
- **Interactive maps** — Zoomable zone maps with multi-level navigation, click-to-place pins, and a saved pin list for marking quest locations, NPCs, or points of interest.
- **Guides** — Linked Text-based Guides and YouTube walkthroughs for dungeon and quest guidance.
- **Search & filters** — Quickly find quests, NPCs, or locations across all dungeons.
